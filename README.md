# 🌤 Weather App (Flutter)

A simple weather application built with **Flutter** that shows current weather information for a selected city.

## 🚀 Features
- 🌡 Displays current temperature
- ☁ Shows weather condition (clear, cloudy, rain, etc.)
- 🌍 Search weather by city name
- 📱 Clean and simple UI

## 🛠 Tech Stack
- Flutter
- Dart
- Weather API (e.g. OpenWeatherMap)
